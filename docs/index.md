## 关于本站

本站是一份**持续维护的前端知识体系**，目标是帮助前端工程师系统梳理技术栈，无论是夯实基础还是追踪前沿。

### 内容原则

- **系统性**：以知识体系为单位组织，而非零散的技巧集合
- **时效性**：标注内容的适用版本和更新时间
- **实用性**：结合实际场景，提供可落地的代码示例

### 如何使用

- 按**模块**系统学习
- 用**搜索**快速定位具体知识点
- 发现问题？点击页面底部「在 GitHub 上编辑此页」提交修正
